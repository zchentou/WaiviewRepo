Acknowledgments

I had to learn Django using Corey Schafer's tutorial on youtube (with 17 videos) as well as a freeCodeCamp.org channel 4-hour long video on youtube about Django (codingforentrepeneurs). Some parts of this code resemble or have ideas that stem from these two, so I will show both of their MIT licenses below.

I also used a variety of open-source third-party packages and different programs to achieve the required functionality throughout my website. This includes django-allauth (for authentication of users), Bootstrap (for formatting of website and HTML), django-crispy-forms (for formatting forms), and even though I didn't end up using it in my final product, I also used Pillow from PIL (to help format images before I switched over to storing them on AWS S3 instead of locally). Instead, I did end up using django-storages (specifically to help with storing images, videos, and various files to AWS S3). I also used gunicorn to help deploy my website using Heroku, as well as django-cleanup to automatically delete the releveant media file from the external storage service in the from of Amazon S3 when a user deletes the respective post or account.
